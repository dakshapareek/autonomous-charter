# 📈 Personal Investment Research Assistant

AI-powered stock research that synthesizes market data, news, and sentiment analysis into comprehensive investment reports.

## ⚠️ IMPORTANT: Install Node.js First!

**Before you can run this app, you need Node.js installed.**

### Quick Install:
1. Download from: **https://nodejs.org/** (click the big green button)
2. Run the installer
3. Restart your terminal
4. Verify: `node --version`

**See [INSTALL_NODEJS.md](INSTALL_NODEJS.md) for detailed instructions.**

---

## 🚀 Quick Start (After Node.js is installed)

**Windows:**
```bash
start.bat
```

**Mac/Linux:**
```bash
chmod +x start.sh
./start.sh
```

## ✨ Features

- Real-time market data via MCP integration
- Multi-source news aggregation
- Reddit sentiment analysis
- AI-powered investment reports
- Risk assessment and recommendations
- Full-stack application (React + Node.js)

## 🏗️ Tech Stack

- **Frontend**: React 18, Lucide Icons
- **Backend**: Node.js, Express
- **AI**: Google Gemini API
